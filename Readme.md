# Midtronic App

## Ejercicio 1;

Este repositorio contiene el algoritmo en la parte de test por si se desea hacer una prueba sin instalar la app (AlgoritmoTest).  
Tambien se integró una opcion en el nav de la app para probar la funcionalidad de manera visible (AlgoritmoActivity).

La lógica que utilice es comparar la letra actual con la siguiente
La idea es que si la letra actual es "a" y la siguiente es "a" también entonces
el count aumenta hasta que es diferente, ahí es cuando guarda la 1era letra y
le concatena el count que va. Así hasta terminar el string



##  Ejercicio 2:  App
Una aplicación Android desarrollada en Java que contiene la pantalla de Perfil donde se muestra información de Felix Soto.
Tambien otra opción del nav de Países consume un listado de países desde una llamada GET.
Los liste dándole diseño tipo card, al darle clic te abre el detalle solicitado en el documento.
